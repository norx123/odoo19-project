Performance Appraisal

This module provides a complete performance appraisal system integrated with Human Resources. It allows organizations to configure appraisal criteria, assign appraisers to employees, calculate performance scores, and generate reports.

Key Features

  • Define multiple performance criteria
  • Configure appraisal periods and evaluation methods
  • Assign appraisers to employees
  • Calculate performance scores
  • Generate performance reports

Performance Appraisal – User Guide

1. Create a New Appraisal

  • Go to the Employee module.
  • Navigate to Performance Appraisal.
  • Click New.
  • Fill in the following details:
    • Appraisal Name: A descriptive name for the appraisal (e.g., "2024 Mid-Year Review").
    • Employee: Select the employee being appraised.
    • Evaluation Period: Define the period for which the appraisal is being conducted (e.g., Jan 2024 - Jun 2024).
    • Appraisal Date: The date of the appraisal.

2. Assign Appraisers

  • Add one or more Appraisers.
  • You can assign multiple appraisers depending on your evaluation process.
    • ⚠️ Important:
    • The Total Allocated Points must equal 100.
    • Distribute the points among all appraisers.

3. Review Appraisers List

  • Click on the “Appraiser Count” smart button.
  • This will display the full list of assigned appraisers.

4. Appraiser Evaluation Process

  • Each appraiser can access their assigned appraisal line.
  • Appraisers can:
    • Review the employee's performance based on the defined criteria.
    • Provide a score for each criterion.

5. Finalizing the Appraisal

After all evaluations are completed:

  • The administrator reviews the results.
  • Click Confirm to finalize the appraisal.
  • After confirmation, no further changes can be made.
